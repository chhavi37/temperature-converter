# 🌡️ Temperature Converter
A simple browser-based temperature converter built using HTML, CSS, and vanilla JavaScript.

## ✨ Features
- Convert Celsius to Fahrenheit
- Convert Fahrenheit to Celsius
- Easy-to-use interface
- Responsive design
- Instant conversion results

## 🛠️ Built With
- HTML5
- CSS3
- Basic JavaScript

## 🚀 How to Use
1. Enter a temperature.
2. Select the conversion unit.
3. Click the **Submit** button.
4. View the converted temperature instantly.
